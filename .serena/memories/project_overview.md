# Project Overview

## Project Name
realtornote (공인중개사요약집)

## Purpose
An iOS educational application for studying for the Korean real estate agent certification exam. The app provides study materials, practice questions, and reference materials to help users prepare for the exam.

## Key Features
- Study notes and summaries for real estate agent certification
- Excel-based content management
- Google AdMob integration for monetization (with donation, full ad, and launch ad units)
- KakaoTalk integration for sharing
- Firebase integration for analytics, crashlytics, messaging, and remote config
- Dark mode UI
